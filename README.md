# Genie
Is a cooking app that allows users to get recipes for dishes they desire. For the the recipes an API(hugginface) was used to collect the recipes. Users can sign in or sign up so they be able to enable favourites features. I used a demo supabase account as a database, it collected the users and favorites.
The is a toddle button for dark mode and light mode. The is also voice assist helping users to get audio assistance.
The is also an admin dashboard that allows admin to check activity.
